# Security Policy

## Scope

This repository contains **research artifacts** accompanying an academic paper. The smart contracts are intended for educational and research purposes. They have not been audited for production deployment.

## Reporting a Vulnerability

If you discover a security vulnerability in the contract logic, verification specifications, or tooling:

1. **Do not** open a public issue.
2. Email the maintainers at the addresses listed in the paper.
3. Include a description of the vulnerability, affected files, and steps to reproduce.
4. We will acknowledge receipt within 72 hours.

## Disclaimer

These contracts are provided as-is for research purposes. Do not deploy them to mainnet with real funds without a professional security audit.
